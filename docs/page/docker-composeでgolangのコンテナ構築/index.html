docker-composeでgolangのコンテナ構築

docker-composeでgolangの実行環境コンテナを構築する

この記事では、golangの実行環境のコンテナを構築する方法を紹介します。

docker-composeコマンドを使うことで、迅速な構築が可能です。

FROM golang:1.15.2-alpine
RUN apk update && apk add git
RUN mkdir /go/src/app
WORKDIR /go/src/app
ADD . /go/src/app

version: "3"
services: 
    app:
        build: . 
        tty: true
        volumes:
            - ./app:/go/src/app

以上です！
